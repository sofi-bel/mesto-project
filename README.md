## Build Setup
Clone the repo

```
git clone https://github.com/sofi-bel/mesto-project.git
```
Install dependencies
```
yarn install
```
Server with hot reload at http://localhost:9000/
```
yarn start
```
Production Build
```
yarn build
```
Development mode
```
yarn dev
```
