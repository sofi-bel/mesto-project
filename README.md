## Build Setup

Clone the repo

```
git clone https://github.com/sofi-bel/mesto-project.git
```

Install dependencies

```
npm install
```

Server with hot reload at http://localhost:9000/

```
npm start
```

Production Build

```
npm run build
```

Development mode

```
npm run dev
```
